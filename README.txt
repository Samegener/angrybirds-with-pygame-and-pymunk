Angry birds:

Hva trenger man for å få programmet til å funke?
Man må ha alle filene i samme mappe

Man trenger bibliotekene pygame og pymunk. Kan innstalleres i terminalen ved å skrive:
pip install pygame
pip install pymunk
De andre biblotekene burde være standard

For å spille:
Dra fuglen bakover som en katapult og sikt på grisene.
Poenget er å få grisene til å forsvinne. Når dette er gjort vil man få valget mellom å starte banen på nytt eller gå til neste bane.
Hvis man trykker på pauseknappen i venstre hjørne eller trukker på ESC knappen til gamet pause. Her kan man restarte gamet, eller fortsette.
